# Ruby 
ruby 2.5.9

# Rails
Rails 5.2.8.1

# Build 
* bundle install
* rails db:create 
* bunlde exec rake db:migrate db:seed 

# Docker
```
docker-compose build
docker-compose up
```

# System Dependencies without Docker
```
* Linux
sudo apt update
sudo apt install ffmpeg
sudo apt install imagemagick

* macOS
brew install ffmpeg
brew install imagemagick
```
